Privacy Policy — Angelo Fileccia Real Estate Content Application

Effective date: 2026-05-16

This application ("the App") is an internal tool used by Angelo Fileccia, Real Estate Advisor, to publish content to social media platforms on behalf of the account owner.

1. Data Collected

The App does not collect data from external users. It is used solely by Angelo Fileccia to publish content to his own connected social media accounts via authorized API access.

2. Data Sharing

The App does not share any data with third parties. OAuth access tokens granted by connected platforms are stored locally and used only to publish content on the account owner's behalf.

3. Data Stored

The App stores only:

  • OAuth access tokens and refresh tokens issued by the connected platforms
  • Content drafts and metadata related to scheduled posts

4. User Rights

The account owner can revoke API access at any time by visiting the connected platform's app permissions settings and removing this application.

5. Contact

Questions about this policy can be directed to: angelo.fileccia@gmail.com
